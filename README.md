# AstraLore — Sitio premium para el lore

Proyecto base para crear una página visualmente impactante destinada al lore de un servidor de Discord. Está pensada para proyectar una estética de producto "multimillonario" con animaciones suaves, tipografía cuidada y una base fácil de personalizar.

Archivos creados:
- [index.html](index.html) — plantilla principal y layout.
- [styles.css](styles.css) — estilos modernos (gradientes, glassmorphism, cursor).
- [scripts.js](scripts.js) — interacciones básicas (cursor, reveal, smooth scroll).
- [assets/logo.svg](assets/logo.svg) — logo SVG placeholder.
- [assets/hero-abstract.svg](assets/hero-abstract.svg) — ilustración decorativa.

Contenido integrado:
- Se ha insertado el lore principal en la sección `BLOQUE NUEVO: EL DÍA A DÍA`.
- Se añadió la `CRÓNICA DE PURG4TORY` con historia (Las Fosas, OLYMPO, etc.).
- Fichas de personajes interactivas con modal: `Luigi`, `Emi`, `Twoky`, `Ozy`, `Nelcon`.
- Se agregó la sección `La lista de ejecuciones públicas` con ejemplos históricos.

Cómo probar localmente
1. Abrir [index.html](index.html) en el navegador (doble clic) o usar la extensión Live Server de VS Code.

Notas rápidas
- Este repositorio es una página estática. No hay backend por defecto.
- Los activos están en la carpeta `assets/`.
- Si necesitas despliegue rápido, GitHub Pages o Netlify funcionan directamente con estos archivos.
